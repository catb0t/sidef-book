# Interactive mode

The interactive mode (aka REPL) is available by simply executing the `sidef` command, or by specifying the `-i` command-line switch.

```shell
$ sidef -i
Sidef 3.05 on linux
Type "help", "copyright" or "license" for more information.
>> n = 41
=> 41
>> n**2 + n - 1
=> 1721
>>
```
